# МП-Сверка — статический сайт для GitHub и Vercel

Готовый проект без сборки:
- `index.html` — продающий лендинг
- `calculator.html` — отдельная страница с калькулятором выгоды
- `styles/main.css` — все стили
- `scripts/config.js` — контакты и базовые настройки
- `scripts/site.js` — поведение лендинга
- `scripts/calculator.js` — логика калькулятора

## Что поменять перед публикацией

Открой `scripts/config.js` и замени:
- `telegramUrl`
- `email`
- `phone`

При необходимости поправь тексты в `index.html` и `calculator.html`.

## Как загрузить в GitHub

1. Создай новый репозиторий на GitHub
2. Залей в него содержимое этой папки
3. Убедись, что в корне лежат `index.html` и `calculator.html`

## Как задеплоить на Vercel

1. Зайди в Vercel
2. Нажми **Add New → Project**
3. Подключи GitHub-репозиторий
4. Никакие build settings не нужны
5. Нажми **Deploy**

Vercel поднимет сайт как статический проект
